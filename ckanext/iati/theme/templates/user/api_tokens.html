The API tokens are encrypted keys (strings) that identifies a user and can be used by applications to connect to the IATI Registry. Because of security reasons, after an API Token is generated, it needs to be copied right away since it won't be accessible again.

If tokens are lost, a new token needs to be generated. There is no limit to how many tokens can be generated for a user.

To revoke access to a token, click the red x button next to the relevant token.
