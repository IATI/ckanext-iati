Welcome to the IATI registry The online repository of IATI raw data links

The IATI Registry contains links to raw data in the IATI XML format. The data itself is not held within the registry, it is published on publishers own websites. The registry simply provides links to this data, and a basic preview function for each data file published. Learn more here.
